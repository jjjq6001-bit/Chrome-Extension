# HBSY VideoGrabber Pro 商店提交指南

## 1. 注册开发者账号

1. 访问 [Chrome Web Store Developer Dashboard](https://chrome.google.com/webstore/devconsole)
2. 使用 Google 账号登录
3. 支付一次性 **$5 美元** 注册费
4. 填写开发者信息

- 文件：`HBSY_VideoGrabber_Pro_v1.0.0.zip`
- 位置：`VideoGrabber/` 目录下

## 3. 提交步骤

### 步骤 1：上传扩展包
1. 登录 [Developer Dashboard](https://chrome.google.com/webstore/devconsole)
2. 点击 "新建项目" 或 "Add new item"
3. 上传 `HBSY_VideoGrabber_Pro_v1.0.0.zip`

### 步骤 2：填写商店信息
1. **商店列表** (Store Listing)
   - 填写名称、描述
   - 上传图标和截图
   - 选择分类和语言

2. **隐私权** (Privacy)
   - 声明扩展权限用途
   - 填写隐私政策（如有）

3. **定价和分发** (Pricing and Distribution)
   - 选择免费
   - 选择发布地区

### 步骤 3：提交审核
1. 检查所有信息
2. 点击 "提交审核" (Submit for Review)
3. 等待 Google 审核（通常 1-3 个工作日）

## 4. 审核注意事项

### 常见拒绝原因：
1. **权限过多** - 只申请必要的权限
2. **描述不清** - 清楚说明扩展功能
3. **隐私问题** - 说明数据使用方式
4. **版权问题** - 不要声称可以下载受保护内容

### 我们的权限说明：
- `storage` - 保存下载任务和设置
- `downloads` - 执行文件下载
- `activeTab` - 检测当前页面的视频
- `host_permissions: <all_urls>` - 访问任意网站检测视频

## 5. 更新扩展

1. 修改 `manifest.json` 中的版本号
2. 重新构建：`npm run build`
3. 重新打包：PowerShell 运行打包命令
4. 在 Developer Dashboard 上传新版本

## 6. 打包命令

```powershell
# 在 VideoGrabber 目录下运行
npm run build
Compress-Archive -Path dist\* -DestinationPath HBSY_VideoGrabber_Pro_v1.0.0.zip -Force
```
