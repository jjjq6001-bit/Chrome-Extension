## 🚀 发布步骤

### 步骤 1: 推送到 GitHub

```bash
# 如果还没有添加远程仓库
git remote add origin https://github.com/YOUR_USERNAME/videograbber-pro.git

# 推送代码
git push -u origin main
```

### 步骤 2: 创建 Tag 触发自动发布

```bash
# 创建版本标签
git tag v1.0.0

# 推送标签
git push origin v1.0.0
```

### 步骤 3: 等待 GitHub Actions

1. 访问 GitHub 仓库 → Actions
2. 等待 "Release" 工作流完成 (约 2-3 分钟)
3. 访问 Releases 页面确认文件已上传

### 步骤 4: 下载发布文件

从 GitHub Releases 下载：
- `HBSY_VideoGrabber_Pro_v1.0.0_Install.zip`

### 步骤 5: 上传到蓝奏云

1. 访问 https://up.woozooo.com/
2. 上传 ZIP 文件
3. 获取分享链接
